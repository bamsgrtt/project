HTML

HTML adalah singkatan dari HyperText Markup Language. Ini adalah bahasa standar yang digunakan untuk membuat halaman web dan menampilkan berbagai konten di browser web. HTML menggunakan tag-tag untuk menandai elemen-elemen dalam sebuah halaman, seperti teks, gambar, tautan, dan lainnya.

CSS

CSS, atau Cascading Style Sheets, adalah bahasa yang digunakan untuk mengatur tampilan dan gaya elemen-elemen dalam dokumen HTML atau XML, seperti halaman web. CSS memungkinkan pemisahan antara konten (HTML) dan presentasi (CSS), sehingga memudahkan dalam pengelolaan dan pemeliharaan tampilan website.
